What is Relaxyz?
Relaxyz is a breathwork timer that helps you build a consistent practice. Choose from four breathing patterns, or create your own. Choose from four soundscapes - so you can close your eyes and still follow the breathing pattern.

Vibed with Google AI Studio. Built on Nostr.
